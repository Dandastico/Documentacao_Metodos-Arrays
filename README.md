# Métodos em JavaScript

Trabalho universitário onde produzi um site explicando alguns métodos presentes em JavaScript.

Os métodos explicados são:
- Array.sort()
- Array.reverse
- Array.toSorted()
- Array.toReverserd()
- Math.min()
- Math.max()

Para construir o site, utilizei HTML, CSS e JS.
